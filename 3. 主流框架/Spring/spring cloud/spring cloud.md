# spring-cloud-commons

大名鼎鼎的SC的commons抽象。含有如：服务注册/发现、熔断器、负载均衡等**公共抽象**，面向该抽象编程可以无需关心底层实现，带来一致的编程感受，这是Spring家族最擅长干的事

> 应该就是定义了一些接口，当作bean来导入，但还需要第三方来实现并注入真正的bean实现类。
