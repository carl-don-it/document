# navicat

常用快捷键
ctrl + q: 打开新查询窗口

ctrl + r: 运行当前窗口内的所有语句

ctrl + w: 关闭当前窗口

ctrl + d: 在表数据窗口上查看表定义

F6: 打开一个MySQL命令行窗口

非常用快捷键

ctrl + n: 打开新查询窗口

ctrl + shit + r: 只运行选中的语句

ctrl + /: 注释

ctrl + shift + /: 取消注释

ctrl + l: 删除一行

F7: 运行从光标当前位置开始的一条完整sql语句
